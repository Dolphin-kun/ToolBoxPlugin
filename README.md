# ツールボックスプラグイン

### [最新バージョンをダウンロード](https://github.com/Dolphin-kun/ToolBoxPlugin/releases/latest)

## 概要
様々な機能をカスタマイズ可能なパネルからボタンで呼び出せるツールを追加するプラグインです。

## 使い方
1. 画面左上のメニューバーから、`ツール`内にある「ツールボックス」を選択してください。

## アイテム一覧（随時更新）
|カテゴリ|名称|機能|
|-|-|-|
|タイムライン|整列|選択した複数のアイテムを、指定した間隔で整列させます。|
|タイムライン|階段状に配置|選択した複数のアイテムを階段状（レイヤーと時間をずらして）配置します。|
|タイムライン|立ち絵配置|音声アイテムに合わせて、立ち絵アイテムを自動で配置します。|
|レイアウト|グループ化|選択した複数のアイテムをグループ化します。|
|テキスト|ルビ振り|テキストアイテムの漢字に自動でルビ（ひらがな/カタカナ）を振ります。|
|テキスト|テキスト分割|テキストアイテムを1文字ごと、または改行ごとに分割して配置します。|
|オーディオ|録音|マイクからの音声を録音し、タイムラインに配置します。|
|オーディオ|音声分離|動画等のアイテムから音声を分離します。|
|ショートカット|ショートカット|「元に戻す」「分割」など、YMM4の標準機能をボタンとして配置できます。|
